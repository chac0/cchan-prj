Why to use HTML5 and CSS3?

After long history of improvement of these markup languages started begging of 1990, it finally came to the point HTML and CSS are used not only for usuability of information viewing, but also for control of web applications.

HTML 5 is the latest version of the HTML language. HTML 5 came out with a lot of improvements over its predecessors, offering better and more innovative ways to meet the increasing demand of sophisticated cross-device and cross-browser websites. HTML 5 is useful in building interactive websites with high-quality graphics and animations.

CSS 3 is the latest version of CSS with a lot of innovative features like multiple tone backgrounds, transition and transformation properties, text-shadow property, rounded corners, box shadows, transparency and much more. CSS 3 gives the web designers and developers, better formatting options for an attractive presentation of web pages.
